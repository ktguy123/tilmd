# TILMD
> Today I Learned MarkDown maker  

Create markdown file for "Today I Learned"  

## Install
```
$ npm install -g tilmd
```

## Usage
```
$ tilmd --help

  Create markdown file for "Today I Learned"
  
  Usage
    $ tilmd [filename] [options]

  Example
    $ ctilmd studyfile -a 

  Options
    --regster -r   Generate a file based on the registered file (default, light, detail)
```
